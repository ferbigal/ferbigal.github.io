Sobre

Interessado e auto-didata em tecnologia e computação desde criança, comecei a me interessar por programação recentemente. Iniciei o curso superior em Análise e Desenvolvimento de Sistemas na FATEC Ipiranga em 2012, onde tive uma base de lógica de programação.

Além disso, busquei sempre por conta própria ampliar meu conhecimento em linguagens de programação por meio de cursos e/ou apostilas. Tenho conhecimento em C#, JavaScript e SQL e também tenho interesse por outras linguagens, como Java, Python, PHP, entre outras.

 

Profissionalmente, trabalhei na WBA Informática de 2012 até 2014, iniciando como estagiário de suporte e em pouco tempo sendo efetivado como Analista Técnico, onde cheguei a ter as seguintes atribuições:

 
  • Treinamento de Software e organização de trabalho em clientes
  • Análise de requisitos e repasse para a área de desenvolvimento da aplicação
  • Criação e adaptação de relatórios via Query SQL
  • Análise de erros do sistema e informações dentro do banco de dados
  • Análise inicial de erros a partir do código do sistema WEB
  • Análise de erros em arquivos XML
  • Configuração e análise de erros na utilização de Certificados Digitais
 

Nesse período trabalhei com tecnologias como: SQL Server (2008 (R2), 2012), SQL Monitor, Certificado Digital e algumas linguagens como: XML, HTML, CSS, JavaScript, JQuery, SQL
